# termy

spawn a pty.js tab with a cwd of file's pwd || project root

### requirements

* [atom](https://atom.io) 1.0

### keybindings

key | action
--- | ------
`ctrl-alt-<enter>` | open terminal in tab. sets the cwd to the current file's pwd, or the project root if no file is open.
